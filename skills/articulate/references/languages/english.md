# English Language Reference

## Focus Areas

### Precise Adjectives

Replace vague modifiers with high-information alternatives. Train the user to reach for the exact adjective instead of the easy one.

| Vague | Precise alternatives |
|-------|---------------------|
| good | compelling, robust, scalable, effective, ergonomic |
| bad | flawed, brittle, untenable, suboptimal, detrimental |
| big | substantial, extensive, significant, sweeping |
| small | granular, incremental, marginal, negligible |
| important | critical, pivotal, foundational, non-negotiable |
| interesting | nuanced, revealing, counterintuitive, noteworthy |

### Strong Verbs

Replace weak verb + adverb combos with a single decisive verb.

| Weak | Strong |
|------|--------|
| help with | facilitate, enable, accelerate, unblock |
| make better | optimize, refine, strengthen, elevate |
| look into | investigate, audit, evaluate, assess |
| get rid of | eliminate, deprecate, sunset, prune |
| put together | assemble, compose, architect, synthesize |
| come up with | devise, formulate, propose, draft |
| go through | audit, review, parse, traverse |
| deal with | resolve, address, mitigate, handle |

### Eliminating Hedging

Hedging weakens professional writing. Train the user to commit.

| Hedging | Direct |
|---------|--------|
| I think maybe we could | We should |
| It seems like it might be | This is |
| We should probably look into | We need to investigate |
| I was wondering if perhaps | Can we / Let's |
| It would be nice if | We need / Requirement: |
| Basically | (delete) |
| Just wanted to | (state the request directly) |
| Kind of / sort of | (use the precise term) |

### Professional Register

Train register-switching across these contexts:

- **Investor pitch**: confident, data-backed, concise. No hedging, no jargon they won't know.
- **Technical documentation**: precise, neutral, complete. No personality, no ambiguity.
- **Customer support**: warm but efficient. Acknowledge, resolve, close.
- **Team communication**: direct, actionable. State the what, the why, the ask.
- **Cold email**: value-first, short, specific. One clear CTA.

### Prompt-Specific Vocabulary

Words that make LLM prompts more effective:

- **Structure**: enumerate, list, categorize, prioritize, rank, compare, contrast
- **Precision**: specify, constrain, define, scope, limit, bound
- **Output**: format, structure, organize, present, render, generate
- **Analysis**: evaluate, assess, synthesize, critique, analyze, audit
- **Behavior**: adopt, assume, act as, emulate, simulate, roleplay

### Business Communication Patterns

Common formats to train:

- Cold emails: hook, value proposition, social proof, CTA
- Follow-ups: reference previous contact, add new value, clear next step
- Proposals: problem, solution, timeline, cost, ask
- Changelogs: what changed, why it matters, what to do
- Release notes: version, highlights, breaking changes, migration path

## Common Weakness Patterns (Non-Native Speakers)

### Article Errors

The most common error for Bulgarian speakers writing English.

- **Missing articles**: "We need to find solution" → "We need to find **a** solution"
- **Wrong article**: "This is best approach" → "This is **the** best approach"
- **Unnecessary article**: "The software engineering is..." → "Software engineering is..."
- Rule of thumb: countable singular nouns almost always need an article

### Preposition Collocations

English prepositions are idiomatic, not logical. Drill these:

- depend **on** (not "from" or "of")
- consist **of** (not "from")
- responsible **for** (not "of")
- interested **in** (not "from" or "of")
- focus **on** (not "at" or "in")
- result **in** (not "with")
- comply **with** (not "to")
- capable **of** (not "to")
- contribute **to** (not "for")

### False Friends from Bulgarian

Words that look similar but mean different things:

| Bulgarian | Looks like | Actually means |
|-----------|-----------|----------------|
| актуален | actual | current, up-to-date |
| евентуално | eventually | possibly, potentially |
| реализирам | realize | implement, carry out |
| конкретен | concrete | specific |
| перспективен | perspective | promising, with good prospects |
| претенция | pretension | claim, demand |

### Register Mixing

Non-native speakers often mix formal and informal registers in the same text.

- **Too formal for Slack**: "I would like to respectfully inquire about the status of the deployment" → "How's the deploy going?"
- **Too casual for proposals**: "So basically we'd kinda fix the thing" → "We propose resolving the data integrity issue through..."
- Train the user to identify the audience and pick ONE register

## Word Lists by Difficulty Level

Use these for FILL_PRECISION mission seeding and daily word selection.

### L1-2 (RECRUIT / INITIATE)

Common precision words that most professionals recognize but underuse:

1. **compelling** — evoking interest or attention powerfully
2. **streamline** — make more efficient by simplifying
3. **robust** — strong, resilient, able to withstand stress
4. **concise** — expressing much in few words
5. **iterate** — repeat a process to approach a result
6. **leverage** — use something to maximum advantage
7. **articulate** — express clearly and effectively
8. **facilitate** — make easier, help bring about
9. **optimize** — make the best or most effective use of
10. **scalable** — able to grow or be adjusted to meet demand
11. **granular** — detailed, broken into fine components
12. **tangible** — clear, concrete, able to be measured
13. **viable** — capable of working successfully
14. **succinct** — briefly and clearly expressed

### L3-4 (OPERATIVE / SPECIALIST)

Register-specific words that separate competent from precise:

1. **synthesize** — combine diverse elements into a coherent whole
2. **enumerate** — list or mention one by one
3. **pragmatic** — dealing with things sensibly and realistically
4. **deprecate** — express disapproval of; mark as obsolete (tech)
5. **orthogonal** — independent, not overlapping (tech/math)
6. **idempotent** — producing the same result regardless of repetition
7. **ergonomic** — designed for efficiency and comfort of use
8. **disambiguate** — remove uncertainty or multiple meanings
9. **deterministic** — producing the same output for the same input
10. **asymptotic** — approaching a limit but never quite reaching it
11. **canonical** — the standard, authoritative version
12. **heuristic** — practical approach, not guaranteed optimal
13. **ephemeral** — lasting a very short time
14. **idiomatic** — natural, native-sounding, following conventions

### L5+ (COMMANDER and above)

Powerful but rare. Use sparingly — only when they are the exact right word:

1. **perspicacious** — having keen mental perception and understanding
2. **parsimonious** — using the minimum resources necessary
3. **verisimilitude** — the appearance of being true or real
4. **recondite** — little known, obscure, dealing with abstruse subject matter
5. **propitious** — indicating a good chance of success, favorable
6. **salient** — most noticeable or important
7. **cogent** — clear, logical, and convincing
8. **efficacious** — successful in producing a desired result
9. **pellucid** — translucently clear in style or meaning
10. **apposite** — strikingly appropriate and relevant
